HTML Character Reference

HTML character references, also known as entities give you an HTML compliant escape for characters

The format is an ampersand, a code reference, then a semi-colon. &entity code; . The code reference could be one of three values:

  • A character name: &copy;
  • A hexadecimal code referring to the Unicode value: &xa9;
  • An "HTML Code" which is a decimal value referring to the Unicode value: &169;
